Metronome LKM is a driver for the Linux kernel (2.6.21 or later) based on high-resolution timers that can generate customizable keyboard events.

Compilation

Create a Makefile with the following content:

obj-m += metronome.o

Then compile the driver using your current kernel headers:

make -C /lib/modules/$(uname -r)/build M=$(pwd) modules
